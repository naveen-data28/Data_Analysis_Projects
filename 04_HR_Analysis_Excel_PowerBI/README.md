**Objective**  
Analyze employee data to evaluate workforce distribution, performance,
and attrition using Excel and Power BI.

**Tools Used**
- Microsoft Excel
- Power BI
- Power Query
- DAX

**Excel Analysis**
- Cleaned HR data and removed duplicates
- Converted datasets into Excel tables
- Created ExperienceYears using hire date
- Built pivot tables for department, salary, gender, and attrition analysis
- Designed an interactive Excel HR dashboard

**Power BI Analysis**
- Imported HR data and performed transformations in Power Query
- Built a star schema data model using EmployeeID
- Created DAX measures for KPIs and ratios
- Designed multi-page dashboards:
  - HR Overview
  - Performance Analytics
  - Attrition Analysis

**Key Insights**
- Department-wise employee distribution and salary trends
- Performance rating analysis
- Attrition rate and employee tenure insights
- Gender and experience-based workforce analysis

**Outcome**
Delivered an end-to-end HR analytics solution combining Excel and Power BI
to support data-driven HR decisions.
